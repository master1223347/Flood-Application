About AquaFour

What is AquaFour?

AquaFour is a project created by high school students with the intention to help communities protect themselves from floods, if they were to occur. We're using both software and hardware in order to accomplish this task.


Software

The programming languages that we used for our project are HTML, CSS, Javascript, Fusion 360 and Python. We used the microweb framework called Flask in order to connect Python with HTML webpages. We also used an API called openweatherAPI to recive the data about the weather. We plugged this data in to recive flood chance(note: it is not 100% accurate as we do not have water level)


Hysense Sensor

The Hysense Sensor is a water sensor that checks the height level of water surrounding it. The water sensor sensor model consists of a solar panel at the top, a gateway at the bottom where the trapezoid is located. The black boxes hold the batteries and the connecter pole holds the wires. How the gate works is it opens up when it detects water, once the water is inside, it holds the water for 2 min and every 2 min it opens and checks if the water level increases or decreases. If it is increased it will send a signal to first responders.

Get your own Hysense Sensor and use our AquaFour app to help alert you in case a flood happens!
